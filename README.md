## Project Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/zsifat/alarm_app
   cd alarm_app
   ```
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
3. **Run the app:**
   - For Android/iOS:
     ```bash
     flutter run
     ```
   - For Web:
     ```bash
     flutter run -d chrome
     ```
   - For Windows/Linux/MacOS:
     ```bash
     flutter run -d windows # or linux, macos
     ```
